# OJ 刷题与学习记录

本仓库主要用于记录个人在算法刷题和数据结构学习过程中的代码。

## 目录结构

- **DataStruct/**: 数据结构学习与实现。
  - `Oredered/`: 线性表结构（链表、栈等）。
  - `Tree/`: 树形结构。
- **HDOJ/**: [杭州电子科技大学 Online Judge](http://acm.hdu.edu.cn/) 的题目解答。
- **luogu/**: [洛谷](https.www.luogu.com.cn/) 的题目解答。
- **Method/**: 常用算法或工具函数的实现（例如 `gcd`）。

## 主要编程语言

- C++
- Python
